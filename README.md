# Volt Pro Website

موقع ويب احترافي لشركة Volt Pro للخدمات الكهربائية والمنازل الذكية.

## 🚀 المميزات

### 🌐 الموقع الرئيسي
- تصميم عصري ومتجاوب
- معرض أعمال تفاعلي
- معرض صور مع تكبير
- دعم كامل للغة العربية
- متوافق مع جميع الأجهزة

### ⚙️ لوحة التحكم
- إدارة المشاريع الكاملة (CRUD)
- تحرير مباشر للمحتوى
- نظام YAML للبيانات
- واجهة سهلة الاستخدام
- حفظ تلقائي للتغييرات

## 🛠️ كيفية الاستخدام

### تشغيل الموقع
1. افتح `index.html` في المتصفح
2. استمتع بتصفح الموقع

### لوحة التحكم
1. افتح `admin.html` في المتصفح
2. أضف/عدل/احذف المشاريع
3. احفظ التغييرات تلقائياً

## 📊 نظام البيانات

الموقع يقرأ البيانات مباشرة من `index.html` باستخدام:
- تحليل تلقائي للمشاريع
- استنتاج الفئات والعملاء
- نظام احتياطي للبيانات

## � نظام الأمان

### مصادقة الدخول
- كلمة مرور بسيطة لحماية لوحة التحكم (قابلة للتحسين)
- جلسات عمل مؤقتة بمدة 24 ساعة
- تسجيل خروج سهل

### نظام النسخ الاحتياطي التلقائي
- نسخ احتياطية تلقائية بعد كل حفظ
- الاحتفاظ بآخر 10 نسخ احتياطية
- إمكانية استعادة البيانات من أي نسخة سابقة

### الحفظ التلقائي
- حفظ تلقائي كل 5 دقائق
- حفظ يدوي في أي وقت
- الحفظ التلقائي قابل للتفعيل/التعطيل

## 🔧 كيفية الاستخدام المتقدم

### تسجيل الدخول
- الدخول إلى `admin.html`
- كلمة المرور الافتراضية: `Volt@2024`
- **تحذير:** غير كلمة المرور في الإنتاج!

### الحفظ والنسخ الاحتياطية
```javascript
// الحفظ اليدوي
admin.saveAllChanges();

// استعادة نسخة احتياطية
admin.restoreBackup(0); // index من النسخ الاحتياطية
```
